# 🎭 Мок-система API для такси

## Быстрый старт

**Сейчас: Фронтенд работает на моковых данных!** 🎉

Все страницы Ayan (такси) полностью функциональны без бэкенда. Данные сохраняются в localStorage.

## Как это работает

### 1. Переключение между моком и реальным API

В файле `frontend/app/config/api.config.ts`:

```typescript
// ⚡️ ПЕРЕКЛЮЧАТЕЛЬ API ⚡️

// Мок режим (разработка фронтенда):
export const USE_MOCK_API = true

// Реальный API (подключение к Laravel):
export const USE_MOCK_API = false
```

### 2. Настройка мока

```typescript
export const MOCK_CONFIG = {
  errorRate: 0.1,      // 10% шанс ошибки (для тестирования)
  baseDelay: 300,      // Минимальная задержка (мс)
  maxExtraDelay: 500,  // Максимальная доп. задержка (мс)
}
```

## Для разработчиков

### Когда готов бэкенд Laravel:

1. Установить `USE_MOCK_API = false` в `api.config.ts`
2. Перезапустить dev сервер: `npm run dev`
3. Всё! Страницы автоматически начнут использовать реальный API

**Не нужно менять код в страницах!** Они используют `useTaxiAPI()`, который сам переключается.

### Тестирование ошибок:

Увеличьте `errorRate` до `0.5` (50%) для проверки обработки ошибок:

```typescript
MOCK_CONFIG.errorRate = 0.5  // Половина запросов будет падать
```

### Сброс мок-данных:

В DevTools → Application → LocalStorage → Clear `mock-api-state`
