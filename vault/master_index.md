## Current Focus

AYAN MVP scope:

1. Create a trip
2. Create a ride request
3. Browse trips and requests with filters
4. Respond and exchange contact details

Out of scope for AYAN MVP:

- maps
- geocoding
- tracking
- push notifications
- ratings
- payments
